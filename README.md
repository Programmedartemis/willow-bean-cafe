# Willow & Bean Café

A modern, responsive café website built with HTML, CSS, and JavaScript.

## Features
- Responsive design
- Sticky navigation
- Mobile hamburger menu
- Smooth scrolling
- Image gallery
- Customer testimonials
- Contact section
- Back-to-top button
- Custom favicon

## Technologies
- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
